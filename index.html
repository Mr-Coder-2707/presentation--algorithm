Overview

StoneSort is an interactive educational web application designed to visualize sorting algorithms in a 3D environment.

  • Educational

    Deep understanding of algorithmic logic.

  • 3D Visualization

    Immersive sorting animations.

  • Interactive Editor

    Write and execute Python code in real-time.

visualizer.py
def bubble_sort(arr):
  n = len(arr)
  for i in range(n):
    for j in range(0, n-i-1):
      # Swap if greater
      if arr[j] > arr[j+1]:
        arr[j], arr[j+1] = arr[j+1], arr[j]

Supported Algorithms

Bubble Sort

Best: O(n) Worst: O(n²)

Simple implementation, good for education.

Quick Sort

Avg: O(n log n) Worst: O(n²)

Fastest in practice, divide & conquer.

Merge Sort

All: O(n log n) Stable

Reliable, stable, but uses O(n) space.

Heap Sort

All: O(n log n) Space: O(1)

Memory efficient, uses Heap data structure.

Insertion & Selection

Insertion: Excellent for small or nearly sorted arrays (Adaptive).
Selection: Minimizes memory writes.
